# SmartFileAI 🔍🤖

SmartFileAI is an offline AI-powered file search engine that allows users to search files and images using semantic understanding instead of keywords.

## 🚀 Features
- 🔎 Semantic text search using transformer embeddings
- 🖼️ Image search using vision models
- ⚡ Fast local indexing (no internet required)
- 🔒 Privacy-focused (fully offline)
- 📁 Supports multiple file types (PDF, DOCX, PPTX)

## 🧠 Tech Stack
- Python
- Sentence Transformers
- FAISS (or similarity search)
- SQLite
- PyQt (GUI)

## 📌 How it works
1. Index files from local system
2. Convert content into embeddings
3. Store embeddings locally
4. Perform cosine similarity search

## 🛠️ Status
🚧 Actively under development

## 🎯 Goal
To build a fast, private, offline alternative to tools like Google search and file explorers with AI capabilities.
