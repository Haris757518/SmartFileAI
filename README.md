# 🔍 SmartFileAI – Offline AI File Search Engine

> ⚡ Fast | 🔒 Private | 🤖 AI-powered | 💻 Fully Offline

SmartFileAI is an advanced offline AI-based file and image search engine that uses semantic understanding instead of traditional keyword matching.

It allows users to search documents, images, and data intelligently — similar to Google Search or Google Photos — but completely offline.

---

## 🚀 Features

- 🔎 Semantic file search using transformer embeddings
- 🖼️ Image similarity search
- 👤 Face-based grouping (in progress)
- ⚡ Fast indexing and retrieval
- 🔒 100% offline (privacy-focused)
- 📁 Supports multiple file formats (PDF, DOCX, PPTX, Images)

---

## 🧠 Tech Stack

- Python
- Sentence Transformers
- PyTorch
- SQLite
- Pillow
- FAISS (planned)

---

## ⚙️ How It Works

1. Files are scanned and indexed locally
2. Content is converted into embeddings
3. Stored in a local database
4. Search queries are matched using cosine similarity
